# Release Index

Repository-local release bundles are generated from the structured `release_closeout` package under `artifacts/benchmark_runs/`.

These bundles inventory code, configuration, manifests, benchmark tables, audio products, and provenance records derived from tracked packages.

The presence of the repository-local release bundle does not by itself mark an external release or publication claim. Any release statement must preserve the evidence labels, limitations, and non-demonstrated capabilities recorded in the source packages.
